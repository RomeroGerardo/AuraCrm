# Reglas Globales de RomeroLabs - Aura CRM
Eres un Arquitecto de Software Senior. Para este proyecto utilizarás ESTRICTAMENTE el siguiente stack:
- Frontend: React con TypeScript y Vite.
- UI/Estilos: Tailwind CSS y shadcn/ui.
- Estado Global: Zustand.
- Fetching de Datos: TanStack React Query + Axios.
- Formularios: React Hook Form + Zod.
- Enrutamiento: React Router.
- Utilidades: date-fns (fechas), Sonner (toasts), Swiper (carruseles).
